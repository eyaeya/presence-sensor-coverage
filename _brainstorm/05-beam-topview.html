三种安装方式 · 顶视图波束足迹模型

每张都是房间顶视。● = 传感器；箭头 = 中轴水平投影；嵌套色块 = 4 个被检测高度的洋葱足迹（站1200 / 坐750 / 躺600 / 地0）。所有足迹都裁到房间内。

吸顶安装

中轴朝正下方。顶视足迹 = 以传感器正下方为圆心的同心椭圆，水平角 φ 只转椭圆朝向。
  • 同心、共朝向，圆心 = 传感器 xy
  • h 越低圈越大（真·洋葱圈）
  • FOV 很大时基本铺满整屋（如实）
  • 传感器可在屋内任意拖拽

侧装

墙内法线 ±ψ 偏航 + 下倾 θ。顶视足迹 = 从墙上一点射入室内的扇形/瓣状区，非同心。
  • 瓣状区从墙上传感器点射入
  • h 越低 → 瓣更大、伸入更远
  • 嵌套但非同心（共墙上根部）
  • 传感器只能沿墙点击/拖拽

墙角安装

沿夹角平分线 45°（固定）+ 下倾 θ。顶视足迹 = 从墙角沿 45° 对角线射入的瓣状区。
  • 瓣状区沿 45° 对角线射入
  • 水平角固定，不可调
  • 嵌套，共墙角根部
  • 点击切换 4 个墙角
站 1200mm 坐 750mm 躺 600mm 地 0mm （实际还会叠加蓝/橙虚线两条探测边界，此图省略以看清足迹形态）
请确认： 吸顶 = 同心椭圆；侧装 = 墙上射入的瓣状区；墙角 = 沿 45° 对角线射入的瓣状区。三者都 "h 越低足迹越大"、都裁到房间内。瓣状形状是示意，实际由椭圆锥∩平面采样精确算出。形态对不对？
