# Independent wall-mounted geometry oracle B

Coordinate: room is `0<=x<=7000, 0<=y<=7000`, mm. `up=(0,0,1)`. Wall bases use local coordinate `s in {100,3500,6900}` along `r=up x n`.

Formula: `a=cos(yaw)n+sin(yaw)r`; `d=cos(theta)a-sin(theta)up`; `u=-sin(yaw)n+cos(yaw)r`; `w=sin(theta)a+cos(theta)up`. For point `P`, `q=P-S`; `forward=q.d`, `side=q.u`, `vertical=q.w`. Inside cone iff `forward>0` and `(side/(forward*tan(hFov/2)))^2+(vertical/(forward*tan(vFov/2)))^2<=1`, plus `|q|<=R` and room clipping.

Point labels per z: `A` axis-z intersection; `H-/H+` horizontal FOV boundary rays; `V-/V+` vertical FOV boundary rays; `CP/CM` distance-circle point along horizontal axis azimuth for presence/motion. `P/M empty` are presence/motion range clipped emptiness flags from independent geometric sampling.
